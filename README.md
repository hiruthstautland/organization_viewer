# Organization Viewer

Project using:

- Auth 0
- Sentry.io
- Github
- react-excel-render

## Usage

Logged in users can upload a excel sheet with org. nr., and get the following information:

- Org. nr.
- Org. name
- Council
- homepage
- Sector code
- Numbers of employed

- Localhost:8080
